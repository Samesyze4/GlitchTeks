Blog Automation Reference

1️⃣ active_blog.html

Purpose: Stores the latest blog post and is dynamically loaded into the community page.

        /blog/active_blog.html

2️⃣ GitHub Action Script (blog_automation.yml)

Purpose: Automates blog updates by pulling new content, archiving old posts, and pushing updates.

        .github/workflows/blog_automation.yml

3️⃣ Blog Generation Script (generate_blog.py)

Purpose: Generates new blog content while avoiding duplicate topics.

        /scripts/generate_blog.py

4️⃣ Archive Management Script (organize_archives.py)

Purpose: Moves old blog posts into an archive and structures them for lookup.

        /scripts/organize_archives.py

5️⃣ Community Page Blog Loader (GlitchBoard.html)

Purpose: Dynamically fetches and displays the latest blog post on the community page.

        fetch("/blog/active_blog.html")
          .then(response => response.text())
          .then(data => {
            document.getElementById("blog-container").innerHTML = data;
          })
          .catch(error => {
            document.getElementById("blog-container").innerHTML = "
Failed to load blog post.
";
          });
